# Changelog

## V1.0 - 2026-05-24

Initial public release of Dongmenlaohu Multi-Agent Engineering Kit.

Included:

- Suite-level multi-agent protocol for Codex, Cursor, and Claude Code.
- Required global rules for interaction, development workflow, and RTK usage.
- Core development skills for brainstorming, debugging, verification, TDD, review, commits, worktrees, architecture, web app testing, database migration, supply-chain audit, and skill creation.
- Fallback task-card templates for light, medium, and heavy tasks.
- Project integration templates for `AGENTS.md`, `CLAUDE.md`, and agent workflow docs.
- Governance docs and scripts for local skill/plugin synchronization.
- Dry-run first bootstrap, verification, diff, rollback, and skill-governance scripts.

Safety boundaries:

- No secrets, local databases, vector stores, project data, or private bootstrap archives are included.
- Installation defaults to dry-run.
- Upstream changes require human-reviewed diff and explicit adoption.
